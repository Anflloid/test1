1. Як створити новий репозиторій?

Зайдіть на сайт GitHub.
У правому верхньому куті натисніть на кнопку + і оберіть "New repository".
Введіть ім'я репозиторію, опишіть його (опціонально) та виберіть тип репозиторію (Public або Private).
Натисніть на кнопку Create repository.
Після створення GitHub надасть вам посилання для клону репозиторію.

2. Для чого призначен репозиторій?

Репозиторій на GitHub служить для зберігання коду, керування версіями проекту, співпраці з іншими розробниками. 
Ви можете ділитися вашим кодом, відкривати Pull Requests, отримувати фідбек від інших користувачів, а також використовувати для зберігання резервних копій вашого коду.

3. Як створити нову гілку у репозиторії?

В GitHub створити нову гілку можна безпосередньо через інтерфейс веб-сайту:
Перейдіть на вкладку "Branches" вашого репозиторію.
Натисніть на кнопку "New branch".
Введіть ім’я нової гілки і натисніть "Create branch".

4. Як злити гілки з основною гілкою master?

На GitHub створіть Pull Request із нової гілки у "master".
Затвердьте Pull Request, і GitHub автоматично злине гілки для вас.

5. Як завантажити репозиторій до себе на комп’ютер?

Потрібно перейти на свій репозиторій
Натиснути на "<> Code"
Натиснути на Download ZIP
